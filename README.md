A simple web application to search for available flights between two airports using the Amadeus Flight Offers Search API. 
Users can input origin and destination IATA codes and select a departure date to view flight details, including flight numbers, airlines, departure and arrival times, and prices.

Features:
Search flights by:
Origin Airport (IATA Code, e.g., JFK).
Destination Airport (IATA Code, e.g., LHR).
Departure Date.
View detailed flight information:
Flight number.
Airline name.
Departure and arrival times.
Ticket price in USD.
Responsive user interface built with React.

Usage
Open the app in your browser at http://localhost:3000.
Enter the origin and destination IATA codes (e.g., JFK, LHR).
Select a departure date.
Click Find Flights to retrieve available flight options.
View flight details, including prices, departure, and arrival times.
